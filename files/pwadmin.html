pwAdmin

- PHP pwAdmin for Perfect World -

Account management, item builder and item webshop for Perfect World game, send item to game character.

  • Main features:
    • Runtime setup wizard with progress bar: mysql connection string, user password encryption type, define the admin user and other basic setups
    • Account management: login, registration, admin tools (delete accounts, user rewarding - points, user info loading and update, send message to the game server)
    • Item builder: ability to choose different additional equipment, options to choose and set up lots of properties, finally the item could be saved as draft, sent to a user playing the game or save it for the webshop
    • Webshop: user can shop items paying them with the gathered points in the game
    • Communication via socket between the game server and the pwAdmin web page
